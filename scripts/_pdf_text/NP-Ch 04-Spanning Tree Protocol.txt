Root Path Cost
15Recall: BPDU contains Root Path Cost
When a switch receives a BPDU, 
it calculates the Internal Root Path Cost of the ingress port
◦Internal Root Path Cost = Root Path Cost (received) + port cost (ingress )
Sum of individual port costs along the best path from the port to Root Bridge
Default port cost defined by the port speed

StateTransition in STP
17When STP is enabled , every port is in a BlockingState
◦Only listening to and processing BPDU, not sending/receiving user frames
Switch Ports remains in a blocking state at election time
Root Port or Designated Port move to a Listening State,
◦All other ports remain in a Blocked State.
Process user frames and update MAC address table 
But does not forward user frames
Forward user frames to the destination
