---
name: e3-config
description: 管理 E3 助手設定（vault 路徑、排除課程等）
---

```bash
node packages/cli/dist/bin/e3.js config get
node packages/cli/dist/bin/e3.js config set vaultPath "/path/to/vault"
node packages/cli/dist/bin/e3.js config set excludedCourses "服務學習,日文,Gender Equity"
node packages/cli/dist/bin/e3.js config set excludedExtensions "mp4,mkv,avi,pkt"
```
