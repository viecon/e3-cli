---
name: e3-open
description: 用預設瀏覽器開啟 E3 頁面
---

```bash
node packages/cli/dist/bin/e3.js open              # E3 首頁
node packages/cli/dist/bin/e3.js open calendar      # 行事曆
node packages/cli/dist/bin/e3.js open security      # 模糊搜尋課程名
node packages/cli/dist/bin/e3.js open 21917         # 用課程 ID
```
